Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. By markup we mean that HTML uses tags to describe the content and structure of a web page. Tags are enclosed in angle brackets, like """ this is in italics""". HTML describes the structure of a web page using elements and tags.

Basic Structure

An HTML document has a basic structure that includes.

  1. DOCTYPE declaration - defines the document type and version of HTML.
  2. html element - the root element of an HTML page.
  3. head element - contains meta-information about the document, like title and links to stylesheets.
  4. body element - contains the content of the document, such as text, images, and links.
